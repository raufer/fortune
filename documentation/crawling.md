# Crawling


Dependencies used:

```
* Google Chrome, v 80
Needed to render pages that have complex JS rendering

* Chrome Driver, v80
It provides capabilities for navigating to web pages, user input, JavaScript execution; sits on top of Google Chrome

* Selenium (python library)
Client API to interact with the chrome driver

* selenium-wire (python library)
Selenium Wire extends Selenium's Python bindings to give your tests access to the underlying requests made by the browser. It is a lightweight library designed for ease of use with minimal external dependencies.
```

```
selenium                    3.141.0
selenium-requests           1.3
selenium-wire               1.0.11
```
